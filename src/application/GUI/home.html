Making Bad is a fake application for buying or selling chemical elements. It is collaborative project by Charles Goodwin and Ryan Merkley.

The GUI is designed by Charles with HTML, CSS, and some JavaScript. The backend uses Java to run PostGreSQL commands to manage a database. JavaFX is used to display the html files and communicate between Java and JavaScript.

The application is currently unfinished, but the portion of the GUI that is complete should function correctly. Some of the remaining features to implement are:

  • completed shopface
  • working shopping cart
  • scripting to fill the user page with their account information
  • manage page for the manager to view the database
  • the Java backend
